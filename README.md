# Signal Mixer

# prerequisites

- [nvm](https://github.com/nvm-sh/nvm)

# installing

```
nvm use
npm install
```

# building

```
npm run build
```

# developing

Changes can be applied in real-time:

```
npm run watch
npm run start:dev
```

# testing

No need to build or run a server:

```
npm run test
```

# usage

```
npm run watch
npm run start:dev
curl localhost:3000/
```
